🎯 Whack-A-Mole Game

A fast-paced, interactive browser game built using HTML, CSS, and Vanilla JavaScript.
Test your reflexes by clicking the mole before it disappears and try to score as high as possible within 30 seconds!

🚀 Features

🎮 Start, Restart, and Play Again functionality

⏳ 30-second countdown timer

🎯 Real-time score tracking

🎲 Random mole appearance every 800ms

🧠 Dynamically generated 3×3 game grid

🔁 Smooth screen transitions (Start → Game → Game Over)

🛠️ Tech Stack

HTML5

CSS3

JavaScript (DOM Manipulation & Timers)

🎮 How to Play

Click the Start button to begin the game.

Moles will randomly appear in different holes.

Click the mole before it disappears.

Each successful hit increases your score.

The game ends after 30 seconds.

Your final score is displayed on the Game Over screen.

Click Play Again to try beating your high score!

🧠 Game Logic Overview

The grid is dynamically generated when the game starts.

A countdown timer runs for 30 seconds.

Moles appear at random positions at fixed intervals.

Clicking a visible mole increases the score.

When time reaches zero, the game stops and displays the final score.
